# IntelliJ Github 연동
## 새로운 프로젝트 만들기
- 파일 > 새로 만들기 > 프로젝트
- 이름 : 프로젝트 이름으로 변경
- Git 저장소 생성 체크
- 샘플 코드 추가 체크 해제
- 생성
## ReadMe.md 파일 생성
- 프로젝트(탐색기) 루트에서 alt+insert(새로만들기)
- 파일
- readme.md
- 여기서 생성된 파일은 GitHub 최초 설명 페이지로 보이게 됩니다.
- 마크다운 문법을 사용하여 작성합니다.

## Git Commit
- 좌측 커밋 탭 클릭 또는 Git > 커밋
- 변경된 내용을 모두 클릭하고
- 커밋 메시지를 작성 후
- 커밋
## GitHub에 프로젝트 올리기

## Git Push
- Git > Push 또는 좌측 커밋 탭에서 커밋 및 푸시
